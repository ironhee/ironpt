# Styles

ironpt has some style of presentation.

You can easily activate style by passing them to Presentation.

#### Types

- `simple`
- `developer`

#### Example: `simple`

```js
<Presentation style={simple}>
  <Slide>
    <H1>Hello world</H1>
    <P>Bla Bla Bla</P>
  </Slide>
</Presentation>
```
