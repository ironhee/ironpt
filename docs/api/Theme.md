# Theme

ironpt has some theme of presentation.

You can easily activate theme by passing them to Presentation.

#### Types

- `simple`
- `developer`

#### Example: `simple`

```js
<Presentation theme={simple}>
  <Slide>
    <H1>Hello world</H1>
    <P>Bla Bla Bla</P>
  </Slide>
</Presentation>
```
